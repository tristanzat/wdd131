Pokéncyclopedia Site Plan

Tristan Zatylny

WDD 131-A1

Overview

Purpose

To help users explore a collection of Pokémon, including stats, types, abilities, etc. through an interactive and responsive web app.

Audience

Fans of Pokémon or those looking to learn more about Pokémon.

Dynamic elements

Each Pokemon (along with its stats and whatever else) will be stored in an object array and displayed dynamically. Filters and searching will be available; content will dynamically adjust according to those.
